
#  Healthcare Management System (Spring Boot)

A full-stack **Healthcare Management System** built using **Spring Boot, Java, MySQL, HTML, CSS, and JavaScript**.
This application allows efficient management of doctors, patients, and appointments with a clean UI and REST-based backend.

---

##  Features

###  Doctor Management

* Add new doctors
* Update doctor details
* View doctor list
* Delete doctor records

###  Patient Management

* Register patients
* View patient details
* Update & delete patient records

###  Appointment System

* Book appointments
* Manage appointment data
* Link patients with doctors

###  Frontend UI

* Simple and user-friendly interface
* Built using HTML, CSS, JavaScript
* Integrated with backend APIs

---

## 🛠️ Tech Stack

* **Backend:** Java, Spring Boot
* **Frontend:** HTML, CSS, JavaScript
* **Database:** MySQL

##  Setup & Run

### 1. Clone the repository

```
git clone https://github.com/darshanaratho/Healthcare-Management-System-SpringBoot.git
```

### 2. Open in VS Code

### 3. Configure Database

* Update `application.properties`

```
spring.datasource.url=jdbc:mysql://localhost:3306/your_db
spring.datasource.username=root
spring.datasource.password=your_password
```

### 4. Run the project

* Run main class: `MngtSystemApplication.java`
